## Как пользоваться

| Колонка | Значения |
|---------|----------|
| **Статус** | `⬜ к выполнению` · `🔄 в работе` · `✅ готово` · `⏸ заблокировано` · `❌ отменено` |
| **MR** | Номер merge/pull request на GitHub (например `#270`) или ссылка |
| **Ответственный** | Имя или @username |
| **Примечание** | Блокеры, ссылки на issues, дата merge |

> Одна задача = один PR (рекомендуется). Крупные задачи (L/XL) можно дробить на несколько PR. Ни одна задача не начинается без согласования соответствующей волны — см. [10-REFACTOR-PLAN.md](10-REFACTOR-PLAN.md).
